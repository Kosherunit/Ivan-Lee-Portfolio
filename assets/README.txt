VIDEO PLACEMENT INSTRUCTIONS

To enable video functionality on the portfolio website:

1. Add your video files to this directory (assets/)
2. Recommended format: MP4 (H.264 codec)
3. Recommended dimensions: 1920x1080 (16:9 ratio)
4. For the main hero section, name your file "hero-video.mp4"
5. For additional portfolio items, name videos logically (e.g., "project1.mp4")

Example video sources in HTML:
- Main hero: <source src="assets/hero-video.mp4" type="video/mp4">
- Portfolio item: <source src="assets/project1.mp4" type="video/mp4">

For optimal performance:
- Compress videos to reduce file size
- Keep videos under 10MB when possible
- Consider using a CDN for larger video files
